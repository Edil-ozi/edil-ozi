## Examples

### Anchor

Use the anchor prop to specify which side of the screen the Drawer should originate from.

The default value is left.

<ComponentPreview name="drawer-demo-anchors" />

## Props

| Prop name | Type           | Default | Description                     |
| --------- | -------------- | ------- | ------------------------------- |
| open      | boolean        | false   | Drawer`s open state             |
| setOpen   | setStateActino | -       | Set open state                  |
| anchor    | string         | left    | Where will be drawer shown from |
