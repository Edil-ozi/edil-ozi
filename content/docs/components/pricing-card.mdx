## Props

### Pricing Card

| Prop name   | Type     | Default | Description                        |
| ----------- | -------- | ------- | ---------------------------------- |
| className   | string   | -       | The className to apply to the card |
| heading     | string   | -       | Heading text of the card           |
| description | string   | -       | Description of the card            |
| price       | price    | -       | Price of the card                  |
| list        | array    | -       | List which will be shown           |
| button      | string   | Click   | Button text                        |
| onClick     | function | -       | Button onClick function            |
