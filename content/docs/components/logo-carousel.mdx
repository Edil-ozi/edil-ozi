### Installation

Copy and paste the following code into your project.

```js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      keyframes: {
        "logo-carousel": {
          from: { transform: "translateX(0)" },
          to: { transform: "translateX(calc(-100% - 2rem))" },
        },
      },
      //Adjust duration as your needs
      animation: {
        "logo-carousel": "logo-carousel 16s linear infinite",
      },
    },
  },
};
```

## Props

| Prop     | Type                             | Description                       | Default |
| -------- | -------------------------------- | --------------------------------- | ------- |
| logos    | `{title: string, url: string}[]` | Data with `title` and `image url` | []      |
| animate? | boolean                          | Whether apply animation or not    | true    |
| shadows? | boolean                          | Whether apply shadows or not      | true    |
